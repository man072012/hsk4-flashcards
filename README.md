# بطاقات HSK4 التعليمية | HSK4 Flashcards

بطاقات تعليمية تفاعلية لمفردات HSK4 مع التصنيف (أفعال / أسماء / صفات)، جمل تطبيقية، ونطق صوتي.

## المميزات

- **تصنيف الكلمات**: أفعال، أسماء، صفات
- **تصفية حسب الموضوع**: المشاعر، الصحة، السفر، الطعام، المال، وغيرها
- **جمل تطبيقية**: جملة بسيطة (HSK4 وأقل) لكل كلمة
- **النطق الصوتي**: باستخدام Microsoft Speech API للكلمة والجملة
- **تتبع التقدم**: علّم الكلمات التي تعرفها والتي تحتاج مراجعة
- **خلط البطاقات**: ترتيب عشوائي للمراجعة
- **اختصارات لوحة المفاتيح**: مسافة للقلب، أسهم للتنقل
- **سحب باللمس**: على الأجهزة المحمولة

## التشغيل

افتح `index.html` في المتصفح مباشرة، أو زر:

**https://man072012.github.io/hsk4-flashcards/**
